Flower Images Classification using CNN

To enhance flower shopping convenience, our system aids users in finding desired flowers through text input and browsing. Users can type flower names to explore options. Additionally, we offer image-based search using ResNet-18 and VGG-16 models, suggesting similar flowers. Our dataset comprises eight types: Baby, Calimero, Chrysanthemum, Hydrangeas, Lisianthus, Pingpong, Rosy, Tana. We evaluated with Logistic Regression and CNN baselines for classification performance and metrics.
